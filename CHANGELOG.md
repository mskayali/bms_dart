## 0.1.0

- Initial release.
- Added full support for JK BMS (JK02_24S, JK02_32S) protocol over BLE.
- Added full support for Daly BMS protocol over BLE, including auto-discovery.
- Built-in multi-frame assembling, checksum verification, and reactive event streams for Telemetry, Device Info, and State tracking.
- Included an example Flutter application with real-time dashboards and battery UI.
